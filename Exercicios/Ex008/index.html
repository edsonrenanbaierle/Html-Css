Principais formatações

Negrito / Destaque

Você pode utilizar a tag b para colocar em negrito, mas não é semantica

Neste formato o modo certo e semantico é uilizar a tag strong pelo fato de ter um significado que seria colocar algo em destaque

Italico / Ênfase

Para colocar em italaco usar tag i (não semantica)

Nesta frase, temos um termo em enfase usando a tag EM (semântica)

Texto Marcado (Marcador de texto)

podemos criar um texto marcado utilizando a tag MARK.

E em outro paragrafo o texto não é marcado com a mesma com de cima, ao menos que como no caso em cima foi delimitada entre o title e o head a css style, fazendo com que todos os mark recebam a mesma cor

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo

Texto deletado

Podemos marcar um texto como exluído para indicar que ele deve ser lido, mas não consierado.

Texto inserido

Podemos colocar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado que utiliza a tag u (não semântica)

Texto sobrescrito

Utilizar para colocar coisas do tipo x2+3 em cima

Texto subscrito

Para inserir coisas do tipo H2O em baixo
